BookShop App

This is a simple bookshop management system. 
It allows users to add books, update stock, and calculate total sales.

Features
- Add books
- Remove books
- View all books
- Calculate total price
- Save data to the database

Technologies used
- Java
- SQL
- HTML/CSS/JS

How to run
1. Download the project.
2. Open it in VS Code.
3. Run the main file.
